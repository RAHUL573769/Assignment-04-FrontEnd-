# 📚 Skill Bridge – Smart Tutor Booking Platforms

A modern, full-stack tutor discovery and booking platform built with cutting-edge technologies. Skill Bridge connects students with qualified tutors through a seamless, intuitive, and visually appealing interface.

---

# ✨ Features

## 🎓 For Students

### 🔍 Smart Tutor Discovery

* Advanced filtering system:

  * Subject / Category
  * Price range
  * Availability (time & day)
  * Featured tutors
* Search functionality with real-time results
* Tutor profile previews with ratings, expertise, and pricing

### 📅 Seamless Booking

* Easy scheduling with interactive calendar
* Real-time availability tracking
* Instant booking confirmation
* Session reminders (planned enhancement)

### ⭐ Review System

* Rate tutors after completed sessions
* Leave detailed feedback
* Transparent rating system to improve trust

### 📊 Personal Dashboard

* View upcoming sessions
* Access session history
* Track payments and activity
* Manage bookings (cancel/reschedule)

---

## 👨‍🏫 For Tutors

### 🧑‍💼 Profile Management

* Create and edit professional profiles
* Add:

  * Subjects taught
  * Experience & qualifications
  * Hourly pricing
* Upload profile images and portfolio details

### ⏰ Availability Control

* Set weekly schedules
* Block unavailable time slots
* Toggle booking status (active/inactive)

### 💰 Earnings Tracking

* Monitor income from sessions
* View analytics (sessions completed, ratings)
* Performance insights (future scope: graphs & trends)

### 📥 Session Management

* Accept or reject booking requests
* Mark sessions as completed
* Track upcoming and past sessions

---

## 🛠️ For Admins

### 📈 Analytics Dashboard

* Platform performance overview
* Metrics:

  * Total users
  * Active tutors
  * Bookings per day/month
* Growth insights (future: charts & exports)

### 🛡️ Moderation

* Review and approve tutor profiles
* Manage reported users or content
* Maintain platform quality

### 👥 User Management

* Control user roles:

  * Student
  * Tutor
  * Admin
* Activate / deactivate accounts
* Handle disputes and support
